Función Sigmoide del Modelo

Transformación de probabilidades del modelo logístico

Cada punto z en el eje x es la suma ponderada de las variables de cliente más un sesgo:

                z = w₁·edad  + w₂·tasa_interes  + w₃·porcentaje_pago  + w₄·dias_de_mora  + w₅·plazo_credito_meses 
                + w₆·num_microseguros  + w₇·n_productos_vigentes  + w₈·n_creditos_vigentes  + w₉·calidad_servicio 
                + w₁₀·estado_ahorro_activo  + b

• Cuando z es muy negativo → σ(z)≈0 → baja probabilidad de deserción (el cliente probablemente se queda).
• Cuando z≈0 → σ(z)=0.5 → incertidumbre (frontera de decisión).
• Cuando z es muy positivo → σ(z)≈1 → alta probabilidad de deserción (el cliente probablemente deserta).

Cada peso wᵢ indica la influencia de esa variable en la probabilidad final. Por ejemplo, un w₄ grande y positivo para dias_de_mora implicará que más días de mora aumentan el riesgo de deserción.
