Fórmula de Regresión Logística

Probabilidad de Deserción:
P(Y=1) = 1
1 + e-z
También conocida como: σ(z) = 1/(1 + e-z)
Donde z es la combinación lineal:
z = β₀ + β₁·edad + β₂·tasa_interes + β₃·porcentaje_pago + β₄·dias_de_mora + β₅·plazo_credito_meses + β₆·num_microseguros + β₇·n_productos_vigentes + β₈·n_creditos_vigentes + β₉·calidad_servicio + β₁₀·estado_ahorro_activo
z muy negativo
σ(z) ≈ 0
Baja probabilidad de deserción
z ≈ 0
σ(z) = 0.5
Frontera de decisión
z muy positivo
σ(z) ≈ 1
Alta probabilidad de deserción
Interpretación de los coeficientes β:
  • • β₀: Sesgo o intercepto del modelo
  • • βᵢ > 0: La variable aumenta la probabilidad de deserción
  • • βᵢ < 0: La variable disminuye la probabilidad de deserción
  • • |βᵢ| grande: Mayor influencia de la variable en la predicción
